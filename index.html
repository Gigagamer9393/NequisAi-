FAQ / Hinweise NequisAi

1. Darf ich den Code weitergeben?

Nein! Der Code ist streng vertraulich und nur für dich bestimmt. Weitergabe kann zum Ausschluss führen.

2. Falscher Code

Der Zugriff wird verweigert. Bitte den korrekten Code eingeben.

3. Mehrfachnutzung?

Nur ein Gerät pro Code erlaubt. Mehrfachnutzung kann sperren.

4. Bannrisiko

Normalerweise Besteht Kein Bannrisiko aber es ist nicht ausgeschlossen!.
